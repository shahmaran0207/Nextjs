# 구현 계획: 네이버 OAuth 로그인 통합

## 개요

이 구현 계획은 기존 JWT 기반 일반 로그인과 NextAuth 기반 네이버 OAuth 로그인을 통합합니다. 네이버 로그인 버튼을 일반 로그인 화면에 추가하고, 네이버 이메일과 일치하는 일반 계정이 있으면 자동으로 연동합니다. 모든 인증은 JWT 토큰을 통해 일관되게 처리됩니다.
