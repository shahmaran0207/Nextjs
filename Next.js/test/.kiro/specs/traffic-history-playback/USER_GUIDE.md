# 교통 상황 재생 기능 사용 가이드

## 기능 개요

교통 상황 재생 기능을 사용하면 과거 특정 시점의 교통 상황을 지도에서 확인하고, 시간 흐름에 따른 교통 패턴 변화를 자동으로 재생할 수 있습니다.

## 패널 열기

1. 디지털 트윈 지도 화면 우측 하단에 있는 **⏱️ 버튼**을 클릭합니다.
2. 교통 이력 패널이 화면 좌측 하단에 나타납니다.

## 주요 기능

### 1. 특정 시점 조회

#### 슬라이더 사용
- 타임 슬라이더를 **드래그**하여 원하는 시점으로 이동합니다.
- 슬라이더 위의 배경색으로 데이터 가용성을 확인할 수 있습니다:
  - **진한 파란색**: 데이터가 존재하는 시간대
  - **연한 회색**: 데이터가 없는 시간대

#### 시간 직접 입력
- **시작 시간** 또는 **종료 시간** 입력 필드를 클릭합니다.
- 날짜와 시간을 직접 입력하여 조회 범위를 변경합니다.
- 시작 시간은 종료 시간보다 이전이어야 합니다.

### 2. 자동 재생

#### 재생 시작
1. **▶️ 재생 버튼**을 클릭합니다.
2. 현재 시점부터 종료 시간까지 자동으로 시간이 진행됩니다.
3. 지도의 교통 상황이 설정된 시간 간격마다 업데이트됩니다.

#### 재생 제어
- **⏸️ 일시정지**: 현재 시점에서 재생을 멈춥니다.
- **⏹️ 정지**: 재생을 멈추고 시작 시간으로 되돌립니다.

#### 재생 속도 조정
- **재생 속도** 드롭다운에서 원하는 속도를 선택합니다:
  - **1x**: 실시간 속도
  - **2x**: 2배속
  - **4x**: 4배속
  - **8x**: 8배속

#### 시간 간격 조정
- **시간 간격** 드롭다운에서 데이터 조회 간격을 선택합니다:
  - **1분**: 1분마다 데이터 조회 (세밀한 재생)
  - **5분**: 5분마다 데이터 조회 (기본값)
  - **10분**: 10분마다 데이터 조회
  - **30분**: 30분마다 데이터 조회
  - **1시간**: 1시간마다 데이터 조회 (빠른 재생)

### 3. 현재 시간으로 복귀

- **🔄 현재 시간으로** 버튼을 클릭합니다.
- 타임 슬라이더가 현재 시간으로 설정됩니다.
- 지도가 실시간 교통 데이터를 표시합니다.
- 재생 중이었다면 자동으로 정지됩니다.

## 지도 색상 의미

교통 상황은 링크(도로)의 속도에 따라 다음과 같이 색상으로 표시됩니다:

- **🟢 초록색**: 원활 (40km/h 이상)
- **🟡 노란색**: 서행 (20-40km/h)
- **🔴 빨간색**: 정체 (20km/h 미만)
- **⚪ 회색**: 데이터 없음

색상 전환은 부드럽게 0.3초 동안 애니메이션으로 표시됩니다.

## 캐시 정보

패널 하단에 캐시된 데이터 개수가 표시됩니다:
- **💾 캐시된 데이터: X개**: 메모리에 저장된 시점 데이터 개수
- **초기화 버튼**: 캐시를 비우고 메모리를 확보합니다.

캐시는 최대 10개 시점의 데이터를 저장하며, 재생 시 끊김 없는 경험을 제공합니다.

## 로딩 및 에러 처리

### 로딩 상태
- 데이터 조회 중에는 **파란색 로딩 스피너**가 표시됩니다.
- 로딩 중에는 슬라이더와 재생 버튼이 비활성화됩니다.

### 에러 메시지
- 데이터 조회 실패 시 **빨간색 에러 메시지**가 표시됩니다.
- 에러 발생 시 이전 상태가 유지됩니다.
- 시스템이 자동으로 재시도를 수행합니다 (최대 3회).

### 시간 범위 검증
- 시작 시간이 종료 시간보다 늦으면 **경고 메시지**가 표시됩니다:
  - "시작 시간은 종료 시간보다 이전이어야 합니다"

## 키보드 단축키

- **Tab**: 다음 요소로 이동
- **Shift + Tab**: 이전 요소로 이동
- **Enter / Space**: 버튼 클릭
- **Arrow Keys**: 슬라이더 미세 조정

## 성능 팁

### 빠른 재생을 위해
- 시간 간격을 **30분** 또는 **1시간**으로 설정
- 재생 속도를 **4x** 또는 **8x**로 설정

### 세밀한 분석을 위해
- 시간 간격을 **1분** 또는 **5분**으로 설정
- 재생 속도를 **1x** 또는 **2x**로 설정

### 메모리 절약
- 사용하지 않을 때는 패널을 닫습니다.
- 캐시 초기화 버튼을 눌러 메모리를 확보합니다.

## 문제 해결

### 데이터가 표시되지 않음
1. 해당 시간대에 데이터가 존재하는지 확인합니다 (슬라이더 배경색 확인).
2. 시간 범위를 조정하여 데이터가 있는 시간대로 이동합니다.
3. "현재 시간으로" 버튼을 눌러 실시간 데이터로 전환합니다.

### 재생이 느림
1. 시간 간격을 늘립니다 (예: 5분 → 30분).
2. 재생 속도를 높입니다 (예: 1x → 4x).
3. 캐시를 초기화하여 메모리를 확보합니다.

### 에러가 계속 발생함
1. 네트워크 연결을 확인합니다.
2. 페이지를 새로고침합니다.
3. 브라우저 콘솔에서 상세 에러 메시지를 확인합니다.

## 접근성

- 모든 버튼과 입력 필드에 **ARIA 레이블**이 적용되어 있습니다.
- **키보드 네비게이션**이 완전히 지원됩니다.
- **스크린 리더**와 호환됩니다.
- 색상 대비가 **WCAG AA 기준**을 충족합니다.

- 데이터 조회 시간: 평균 **1-2초**
- 캐시 히트율: 재생 중 **80% 이상**
- 지도 렌더링: **30fps 이상** 유지
- 메모리 사용량: 최대 **100MB**
